# Analyst_Tasks
Тестовые задания по аналитике.

В этом репозитории собраны тестовые задания по аналитике.
В рамках выполненных заданий:

* смоделированы бизнес-процессы в нотации BPMN 2.0
* выполнено описание бизнес-логики процессов
* спроектирован REST API
* проработаны backend-процессы

### 1. Equipment Request and Fulfillment Process Modeling

* описано взаимодействие между пользователем, технической поддержкой, снабжением, логистикой и поставщиком
* отражена работа нескольких систем
* смоделированы альтернативные сценарии (наличие / отсутствие оборудования)
